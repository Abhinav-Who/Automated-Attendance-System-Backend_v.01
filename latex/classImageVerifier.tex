\doxysubsection{Detailed Description}
Provides functionality to verify user images by comparing them with stored images. 

The \doxylink{classImageVerifier}{Image\+Verifier} class interacts with a database to fetch stored images and compare them with user-\/provided images, returning a similarity score.
